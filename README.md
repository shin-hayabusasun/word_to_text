# Word文書テキスト変換ツール

このツールは、マニュアル集に含まれるWord文書（.docと.docx）をテキストファイル（.txt）に一括変換するためのものです。

## 前提条件

- Python 3.6以上
- Microsoft Wordがインストールされていること（.docファイル変換時に必要）

## セットアップ

1. 必要なライブラリをインストールします：

```
pip install -r requirements.txt
```

## 使用方法

### コマンドライン版

```
python word_to_text_converter.py <マニュアル集のディレクトリパス>
```

これにより、指定したディレクトリ内のすべてのWord文書（.docと.docx）が同じディレクトリ内のテキストファイル（.txt）に変換されます。変換はサブディレクトリも含めて再帰的に行われます。

#### オプション

- `--no-recursive`: サブディレクトリを再帰的に処理しない場合に指定します。

```
python word_to_text_converter.py <マニュアル集のディレクトリパス> --no-recursive
```

### GUI版

より使いやすいGUIバージョンも用意しています：

```
python word_to_text_gui.py
```

GUI版では、以下の操作が可能です：

- ディレクトリの参照・選択
- サブディレクトリを含めるかどうかの選択
- 変換の進捗状況の表示
- ログの表示
- 処理のキャンセル

## 注意事項

- 変換されたテキストファイルは、元のWord文書と同じ場所に作成されます。
- すでに同名のテキストファイルが存在する場合は上書きされます。
- .docファイルの変換には、Microsoft Word（COM）を使用します。
- テキスト変換では書式情報は失われます。表や画像などの要素も変換されない場合があります。
